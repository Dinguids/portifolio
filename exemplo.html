HTML Semântico

O HTML semântico é uma abordagem de escrita de código HTML que visa atribuir significado semântico a cada elemento. Isso não apenas ajuda os desenvolvedores a entender a estrutura do documento, mas também é útil para mecanismos de busca e leitores de tela.
